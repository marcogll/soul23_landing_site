# Noire Server

Portfolio website con modo dark/light, diseño responsive y animaciones.

## Características

- Diseño responsive (Mobile First)
- Tema claro/oscuro
- Animaciones con ScrollReveal
- Filtro de portafolio con MixItUp
- Slider de testimonios

## Estructura

```
noire_server/
├── assets/
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── main.js
└── index.html
```

## Uso

Simplemente abre `index.html` en un navegador.
